🌿 Terminal Git Commands

💻 通常コマンドへ

ステータス
git status
ステージング
git add
コミット
git commit -m "Initial commit"
リモートへ送信 (Push)
git push
git push origin main
git push -u origin main
git push origin main2
リモートの最新を取得
git pull
ブランチ一覧表示
git branch
新しいブランチ作成 & 移動
git checkout -b new-branch
既存ブランチに切り替え
git checkout main
ブランチをマージ
git merge feature
ブランチ削除
git branch -d old-branch
リモートURL確認
git remote -v
リモートURL変更
git remote set-url origin https://github.com/USERNAME/NEWREPO.git
リモート追加 (remote add)
git remote add origin https://github.com/USERNAME/REPO.git
変更内容を確認 (diff)
git diff
履歴をきれいに表示 (log)
git log --oneline --graph
クローン
git clone
リポジトリ初期化 (init)
git init
ファイルを削除 (git rm)

⚡ Git管理下のファイルを削除し、次のコミットで反映される。

❗ ワークツリーからも消えるので注意。

git rm file.txt
変更を元に戻す (git restore)

⚡ ファイルの編集を破棄して最後のコミット状態に戻す。

❗ 編集内容は消えるので取り消せない。ステージ解除には --staged を使う。

git restore file.txt
ステージを取り消す (git reset)

⚡ git add を取り消す。作業内容は残る。

❗ --hard を付けると変更も全て消えるので要注意。

git reset HEAD file.txt
コミットを取り消す (git revert)

⚡ 特定のコミットを「打ち消すコミット」を新規作成する。

❗ 履歴は壊さず残るので、チーム開発では reset より安全。

git revert HEAD
