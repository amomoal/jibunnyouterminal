🌿 Terminal Essential Commands

💻 gitコマンドへ

カレントディレクトリを表示 (pwd)
pwd
ファイル一覧 (ls)
ls
隠しファイルも表示 (ls -a)
ls -a
フォルダに移動 (cd)
cd folder
1つ上に戻る (cd ..)
cd ..
ホームに戻る (cd ~)
cd ~
フォルダ作成 (mkdir)
mkdir new-folder
ファイル削除 (rm)
rm file.txt
フォルダ削除 (rm -r)
rm -r folder
ファイル内容を表示 (cat)
cat file.txt
長いファイルを閲覧 (less)
less file.txt
カレントディレクトリをFinderで開く
open .
ファイルをデフォアプリで開く
open file.txt
